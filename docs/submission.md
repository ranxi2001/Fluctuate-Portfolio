## 📤 提交流程

### HackQuest 平台提交步骤

1. **注册账号**
   - 访问 HackQuest
   - 注册并登录

2. **创建项目**
   - 点击"Submit Project"
   - 选择 Mantle Global Hackathon 2025

3. **填写信息**
   - 项目名称
   - 描述
   - 选择赛道

4. **上传材料**
   - GitHub 链接
   - Demo 链接
   - 视频链接
   - Pitch Deck

5. **提交审核**
   - 检查所有信息
   - 提交

## 💡 提示和建议

### 视频录制技巧
- 使用 OBS Studio 或 Loom
- 提前写好脚本
- 多录几遍选最好的
- 添加背景音乐增加氛围

### Pitch Deck 设计
- 使用 Canva 或 Figma
- 保持视觉一致性
- 图片比文字更有说服力
- 突出核心亮点

### Demo 优化
- 确保快速加载
- 准备测试账户和数据
- 错误处理友好
- 添加引导说明
