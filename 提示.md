🚀 下一步：在服务器上开发
第一步：环境检查
在服务器上运行：

bash
# 进入项目
cd Fluctuate-Portfolio/contracts
# 检查 Node.js 版本（需要 >= 18）
node --version
# 检查 npm 版本
npm --version
# 如果版本过低，升级：
# curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
# sudo apt-get install -y nodejs
第二步：安装依赖
bash
# 在 contracts 目录下
npm install
# 预计安装时间：1-3 分钟
第三步：配置私钥
bash
# 复制模板
cp .env.example .env
# 编辑环境变量
nano .env
# 填入：
# PRIVATE_KEY=0x你的MetaMask私钥
# 保存并退出（Ctrl+X, Y, Enter）
⚠️ 如何获取私钥：

打开 MetaMask
点击账户详情
导出私钥
确保使用测试账户，不要用主网私钥！
第四步：获取测试网 MNT
方法 1：官方水龙头

访问：https://faucet.testnet.mantle.xyz
输入你的地址
领取测试 MNT
方法 2：Discord

加入 Mantle Discord
在水龙头频道请求

🐛 可能遇到的问题
问题 1：npm install 失败
bash
# 清理缓存
npm cache clean --force
rm -rf node_modules package-lock.json
npm install
问题 2：Gas 不足
Error: insufficient funds for gas * price + value
解决：从水龙头获取更多测试 MNT

问题 3：私钥格式错误
Error: invalid private key
解决：确保私钥以 0x 开头，64 个字符

问题 4：网络连接问题
Error: could not detect network
解决：检查 RPC 地址是否正确

Mantle Testnet: https://rpc.testnet.mantle.xyz
